CSS Introduction
  1. CSS stands for (Cascading Styel Sheets)
  2. CSS is converts the html web page into a colourfull and desined webpage
  3. CSS workes in three different ways
  • Inline:- Styles written directly in a HTML tag
  • Internal:-Styles written inside tages in html file
  • External:-Styles written in a separate css file linked to the html file

What is selector ?

1. selector is a pattern used to select and target HTML elements in order to apply styles to them. It determines which elements on a web page will be affected by the CSS rules.

  • Selector diffrentente types
  • Universal selector(*)
  • class selector (.classname)
  • Id selector (#idname)
  • Child selector (div>p)
  • Group selector (comma -separated list like h1,h2,p)
